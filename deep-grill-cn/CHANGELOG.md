# deep-grill / deep-grill-cn Changelog

## 2026-06-07 — v2: 第一性原理从"阶段"变成"可注射模式"

### 问题

旧版将第一性原理锁定在阶段一（解构）和阶段三（重建），是一个线性流水线：

```
解构(FP) → 交叉审问(Socratic) → 重建(FP) → 元审视
```

实际会话中，AI 在阶段二、阶段四、甚至阶段三"重建"时，可能正在复用未经审视的假设——而旧版没有给 AI 切回第一性原理的机制。

### 变更

| 旧 | 新 |
|---|---|
| 4 个线性阶段，FP 专属阶段一和阶段三 | 4 段弧线作为方向，FP 可在任意点注射 |
| "阶段一：解构（第一性原理）" | "模式 B：第一性原理拆解"——不锁在任何阶段 |
| 无切换机制 | 6 个 FP 注射触发条件，AI 自行判断 |
| 对话节奏图是单向箭头 | 网状可回跳结构 |
| 5 条核心规则 | 8 条——新增 #7（模式 B 不是仪式）、#8（宁可多拆一层） |
| 重建段无特殊警告 | 重建段明确标注"最容易踩触发条件 #1" |

### 新增的 6 个 FP 触发条件

1. 拿旧框架的零件拼新东西
2. 结论下得太快
3. "显然""大家都知道"出现
4. 用户说"我没看懂"或"为什么"
5. AI 复用了上一阶段的结论而不加验证
6. 苏格拉底审问转了三轮没进展
